Інструкція

Для початку роботи з сервісом для оформлення бібліографічних посилань та бібліографічного опису необхідно перейти до однієї з вкладок "Посилання" чи "Бібліографічний опис", за допомогою навігаційної панелі або натисниту на один з пунктів "Створити посилання" чи "Оформити опис" на початковій сторінці.
Після переходу до однієї з вкладок, у правій частині необхідно обрати тип наукової роботи, на яку здійснюється посилання чи створюється бібліографічний опис, кількість авторів або укладачів та мову перекладу (оригінальну мову на якій написана наукова робота).

Обрав необхідну категорію для оформлення бібліографічного посилання чи опису, на екрані з’являться поля для заповнення інформації про роботу, наприклад: якщо ви обрали книгу з двома авторами, то необхідно заповнити інформацію про ПІБ авторів, назву наукової роботи (книги), тип роботи, видавництво, місто та рік видання, кількість сторінок, за редакцією та, для бібліографічного опису, нотатки.
Для оформлення депонованих наукових робіт, неопублікованих перекладів, перекладених ресурсів оберіть тип джерела "Книга" та доповніть поле з нотатками необхідною інформацією.
Після того, як необхідні поля будуть заповнені, необхідно натиснути на кнопку «Створити посилання (опис)». Оформлене посилання та транслітерація буде відображено у блоці під назвою «Бібліографічні посилання (Бібліографічний опис)».

Наявна можливість завантажити створенні посилання або зберегти у необхідному форматі для статті, також окреме посилання можно скопіювати або видалити за непотреби чи видалити усі створені посилання.
Створенні посилання зберігаються у таблиці після закриття сторінки браузера або самого браузера. Для зручної роботи з великим обсягом раніше створених посилань, вбудовано такі можливості, як:

  • відображення створиних посилань у обсязі від 5 до 20 рядків у таблиці;
  • можливість пересування між початковою та останньою сторінкамі таблиці;
  • пошук посилань за ключовими словами за рядком "Пошук посилань/за бібліографічним описом";
  • сортування створених посилань за алфавітом.

ONAFT4Ref відповідає стандарту ДСТУ 8302:2015 та оформлює бібліографічні стандарти по наведеному прикладу.
